# S3 Security Research

My hands-on research into AWS S3 security vulnerabilities 
and misconfigurations.

## What's Here
- Lab observations (7 hands-on labs)
- External testing notes
- Bug patterns discovered
- FIND-FOCUS-FIRE framework applied to S3

## Research Scope
Day 1-5 of my security learning journey. Testing S3 
permissions, understanding authorization, and preparing 
for bug bounty hunting.

## My Approach
Learning by doing — hands-on labs, external testing 
with AWS CLI, and documenting everything.

## Key Finding
S3's stated purpose is secure object storage. 
The actual behavior reveals permission misalignments 
that most hunters overlook.
